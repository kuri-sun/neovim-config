# Neovim config

### 1. Location 
 - Mac, Linux <br />
  HOME/.config/nvim
 - Windows <br />
  

### 2. Packer install in your local machine.

 Mac, Linux

```
git clone --depth 1 https://github.com/wbthomason/packer.nvim\
 ~/.local/share/nvim/site/pack/packer/start/packer.nvim
```

 Windows

### 3. you can do :so in plugin.lua, and run :PackerSync.
